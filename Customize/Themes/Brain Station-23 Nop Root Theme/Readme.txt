1. 'NopRoot' theme is released for nopCommerce 3.20 and nopCommerce 3.30
1. 'NopRoot' directory contains views and contents.
2. To activate 'NopRoot' theme copy that 'NopRoot' folder of the particular version and paste it to the 'Themes' folder of your server.
3. Go to admin panel of your site and go to Configuration -> Settings -> General and Miscellaneous Settings.
4. Select 'NopRoot' among the 'Desktop store theme' options and save.
